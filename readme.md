# UTF-8 Game Engine

This is a JavaScript / TypeScript library for building web browser games entirely out of UTF-8 text characters.

Because... Why not? 🤷‍♀️

```
npm i utf-8-game
```

```javascript
import * as u from 'utf-8-game';
```
